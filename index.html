SELECTearch

A simple, sweet and clean jquery plugin to convert your "SELECT BOX" into Searchable Select Box.!

View project on GitHub

Intro

SELECTearch - a plugin to make out most of simple and normal select box. Add this into your project and turn your select boxes into seacrhable select boxes.

Usage

It is as simple as 1,2,& 3.. All you need is:

1. Latest jQuery included in your page.

2. Obviously, include selectearch.js after jQuery.

3. Some CSS that goes in head tag, to beautify the things :

<style type="text/css">
* {
    font-family:"Trebuchet MS", Arial, Helvetica, sans-serif;
    font-size:100%;
    border:1px solid rgba(0,0,0,0.2);
    padding:0.5em;
    border-radius:8px;
    box-sizing:border-box;
    outline:none;
}

html,body {
    padding:0px;
    border:0;
}

option {
    border:0;
    font-size:1em;
    padding:0.25em 0.5em;
}

select {
    display:block;
    width:25%;
}

label {
    border:0;
    display:block;
    width:100%;
}

input {
    width:100%;
    display:block;
}

.selectearch {
    border-top:0;
    border-bottom:0;
    border-radius:0;
}

.selectearch_search_box {
    border-bottom:0;
    border-radius:8px 8px 0 0;
    box-shadow:inset 0px 1px 3px rgba(0,0,0,0.2);
}

.selectearch_results {
    border-top:0;
    border-radius:0px 0px 8px 8px;
}
</style>

4. A Select element, something like below:

<select>
<option value='0'>Apple</option>
<option value='1'>Samsung</option>
<option value='2'>Nokia</option>
</select>

5. And selectearch(); before closing your body tag :

<script type="text/javascript">
$(document).ready(function() {
    $("select").selectearch();
});
</script>
